交易系统

发表于 2022-11-20 更新于 2022-11-24 分类于 交易系统

感谢茶话股经的启蒙教育与背离、均线分析

感谢无知青年刘敬梓、刀哥、付鹏的基本面分析理论

感谢永乐大帝明成祖的中枢理论以及某匿名知乎用户的突破理论

我交易系统的主框架已基本形成

核心

买入突破中枢上沿的标的，做空下穿中枢下沿的标的

具体操作

买点

一级买点

一级买点为可以直接重仓介入的最佳买点

突破回踩

突破后立即回踩上边界，或者上涨后的一次回踩都视为机会

中海油服10.24后的两个交易日，中国稀土11.16之后的几个交易日是典型的机会

对应止损位：突破点向下1-5个点

新中枢下沿

招轮的5月、7月、当前的11月都算是可行的长周期买点

对应止损位：中枢下沿向下1-5个点

卖点

在这里，卖点指止盈点

一级卖点

一级卖点为应考虑大幅减仓（50-100）的卖点

突破失败

在当前操作级别，某根K线逼近中枢上届或长周期压力位，发生次级别逆转时，应考虑大幅减仓甚至清仓

历史上招轮8.2日和海能9.6日等都产生过类似卖点

不过，减仓后续应考虑回补，参考回补点位：次级别横盘、次级别背离、次级别中枢下沿等

二级卖点

二级卖点为应考虑适当减仓（30-60）的卖点

注意，二级卖点后续需要设置右侧接回的点位，同时回补标准应相比一级卖点降低

收敛楔形上边界

在高级别中枢中，次级别可能表现为楔形收敛，其对应上边界可以考虑主动减仓

减仓后续应考虑回补，参考回补点位：次级别横盘、次级别背离、次级别中枢下沿等

大幅拉升后的背离与逆转

短期15%以上的拉升后，可能会有调整，可以考虑在背离与逆转后减仓

8.2日附近的招轮是典型的例子

具体情况说明

中国稀土-2022-11-24

可行的操作方案有两种

如果较为看好的话，等实体跌破33的第二根K卖出

反之，则可以在今天9.40-9.45的两根线上伺机卖出

还有一点教训是：（A股）破位清仓的票，当日盘中不能回补。尾盘视情况可以回补

Java Annotation

Annotations, a form of metadata, provide data about a program that is not part of the program itself. Annotations have no direct effect on the operation of the code they annotate.

Annotations have a number of uses, among them:

  • Information for the compiler — Annotations can be used by the compiler to detect errors or suppress warnings.
  • Compile-time and deployment-time processing — Software tools can process annotation information to generate code, XML files, and so forth.
  • Runtime processing — Some annotations are available to be examined at runtime.

yaml

YAML是一个可读性高，用来表达数据序列化的格式。YAML参考了其他多种语言，包括：C语言、Python、Perl，并从XML、电子邮件的数据格式（RFC 2822）中获得灵感。Clark Evans在2001年首次发表了这种语言.

YAML是”YAML Ain’t a Markup Language”（YAML不是一种标记语言）的递归缩写。在开发的这种语言时，YAML 的意思其实是：”Yet Another Markup Language”（仍是一种标记语言），但为了强调这种语言以数据做为中心，而不是以标记语言为重点，而用反向缩略语重命名。
