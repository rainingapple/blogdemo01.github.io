2022-5-29-交易日志

发表于 2022-05-29 更新于 2022-05-31 分类于 交易日志

距离上次4.28交易日志已经过去了近一个月，期间忙于论文投稿，无空盯盘，一直保持空仓。

前言

美股

  • 长期走势：虽然目前外围对于危机的传言甚嚣尘上，但是从道指和纳指周月季走势来看，其依旧保持着完美的多头趋势，超长期走势具体的走向要看万点附近方向的选择。单纯从技术上来看，应该会在3-6个月内做出选择。
  • 中期走势：日周线级别目前还是保持着完美的空头走势，但是没有走出2008、2020年类似的恐慌式下跌。但是空头最强的阶段很可能已经过去，中期下跌走势很可能已经过半，结合短期走势来看，有希望在两周内逐步震荡摸底展开中期反弹。
  • 短期走势：日线MACD金叉，30分钟接近死叉，短线应该会有1天左右的调整。但是整体的短期反弹没有走完，后续的强压力位在季线。

港股

  • 长期走势：03、08、15形成的季线通道已被破坏，季线长期走势混沌。月依旧是空头趋势，有望实现底背离。虽然后几年是否会继续杀跌尚不清楚，但是就短期3-5个月来说，震荡或震荡向上可能较大。
  • 中期走势：周线不标准底背离，日线二次底背离后金叉，空头趋势不足。所有短期均线与季线贴合，非常有望展开目标年线的中期行情。
  • 短期走势：30、60金叉区间，5、15macd绿柱明显缩短，短期走势不是特别明确。
  • 应密切关注，可以适当仓位参与，也可以等底背离后加仓。

A股

  • 长期走势：上证年线级别依旧在上升通道内，季线在08-15收敛三角形上沿。如果这一根季线最终能收于3100之上，趋势就没有破。
  • 中期走势：周线逼近250周线压力位，日线逼近季线压力位，中期反弹基本到了尾声。
  • 短期走势：中期风险逼近，短期走势失效。
  • 强压力位结合走势可以做空

主要指数

上证50 沪深300

  • 季线上目前已经跌破15-18形成的上涨通道，这跟季线能否站上2900（最好3000）将直接决定今年下半年的权重行情。
  • 周线接近底背离，日线背离后底部震荡。中期来说，在技术面与消息面的共同作用下有望展开向年线的反弹。具体时机需要结合30分钟级别走势。
  • 低级别走势关注背离点

  • 中期反弹接近尾声，可以考虑做空

2022-4-19-交易日志

发表于 2022-04-19 分类于 交易日志

关于后面的路

  • 近期偶然的机会让我获得了足够的本金，不过仔细地审慎自身，却发现我还是没有驾驭的力量。回想年后大概两个月的交易，有得有失。今天借这个机会自然应该好好反省，我的交易体系以及交易心态成功在哪里，哪里又有着缺陷。

  • 这两个月在对上证的大方向的判断上完全正确，也因此成功躲过了三月的暴跌。成功把握了建筑园林的最强势行情以及后续部分双顶行情，部分参与了粮食的炒作。个股操作上，我认为只有交建股份的操作可圈可点，节奏完全正确。

  • 不过更重要的是，我究竟因为什么亏了钱？

    • 首先是大盘，虽说我选取的板块个股大多是走着自己的日线多头趋势，但是在大盘整体不稳（尤其是外围造成的不稳）情况下，个股的走势会波动更大同时更为弱势，最关键的是我会在外围影响下怀疑自己的买入逻辑，造成失败变形的操作。这里有一个本质上的二元对立，在我的交易体系中，外围不佳应该直接清仓，同时强势板块回调十日线又该拿住，这两点叠加起来整体就是矛盾的，自然也会造成畸形的操作。就这一点上，我认为我应该做的有两点：

      • 更严格的审视自己的买入计划，不严格满足直接空仓，买入后逻辑不破要坚定持有
      • 严格控仓，在大盘走出可能反转横盘之前，仓位不超过15000，平时不超过10000
      • 强外围利空一旦发生依旧无条件卖出

    • 其次是买点，我想主要分两点来说：建仓以及补仓。

      • 一波日K的上涨趋势，强度有强有弱，并且开始的时机和大盘、板块走势、主力资金、消息面都有关系，如何去参与其实是一个非常复杂的话题，并且很难说有一套万能公式，我们只能去做一个概率。30分钟以上的金叉、背离、双底都有可能是一波上涨的起点。
      • 建仓点没有那么明确也不可能太精准，左侧的话，在板块或者个股经历充分调整后，找一个合适的价格合适的买点问题都不会太大。右侧的话，在单日中阳拉上去之后，找一个分钟买点进入一般也不会被套（大概3天以内）
      • 但是什么情况危险是很明确的。板块连续上涨（中间经历过短时回调）后首次破5不能着急建仓，要等待背离和十字星；顶背离后不能着急建仓，至少要等macd死叉阶段结束。日内拉升一定幅度后，追要谨慎。
      • 因为我的补仓成功率极低，所以暂时禁止该项操作

    • 其次是仓位，总体仓位的控制在大盘那一项已经提到过了，这里不再赘述。就个股而言，主要谈两个操作，浮盈加仓和浮亏加仓，无论从结果还是逻辑上来看，这种操作都不好，暂时都应禁止。一次买入、一次卖出。

  • 深证与创指短期来看确实超跌，同时形成120分钟底背离，但是这波能到哪里不得而知，也不排除直接震荡
  • 上证50目前位置很有可能继续下行，中证1000有机会展开数天的超跌反弹
  • 小盘股走势可能强于大盘一段时间，指数很可能也只是微跌，但是大的风险近在咫尺！
