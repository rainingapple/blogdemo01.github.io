PythonLearn-03-函数与异常

发表于 2020-10-21 分类于 Python

函数

关键字参数与位置参数

顾名思义，xy为位置参数，z为关键词参数

注意，位置参数必须在关键词参数之前

  File "<ipython-input-2-cd92909de81c>", line 1
    def fuc(x=1.6,y,z=1.5):
            ^
SyntaxError: non-default argument follows default argument

不难想到，它特别要求这个里面的位置参数必须在关键字参数之前，是为了保证位置传参的规范性与关键词传参的灵活性

函数是对象

这里我想说的是，这个特性要比Java还要灵活。具体来说，我甚至可以对函数进行遍历，这是一个难以想象的技术。

['Spspp', 'Dddefe']

感觉oj拿这个写是真的爽，报时就不爽了。。。

同样的我们甚至可以把函数作为参数传给函数，比如map函数

['Spspp', 'Dddefe']

匿名函数

按照书上的意思是，它不仅代码量小，而且简单易懂结构清晰

[16, 36, 49, 81]

生成器(Generator)

正如前文经常遇见的，python有一种特性叫可迭代。

这个特性其实是由迭代器协议实现的，即令对象可遍历的通用方式

迭代器就是一种向python解释器生成对象的对象

<dict_keyiterator at 0x2491ddbef40>

生成器是构建可迭代对象的一种非常简洁的方式，普通函数执行并一次返回多个结果

而生成器则惰性的返回一个多结果序列，即产生之后暂停直到下一个请求

如需构造一个生成器，只需将return换为yield

并且要注意，生成器生成完了就啥都没了，不要重复调用

还有就是到这里才明白了，为什么有一些自己不输出必须要到list调用才输出。
